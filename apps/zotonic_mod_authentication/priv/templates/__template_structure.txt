SIGN IN
=======

logon.tpl // log on page
|-- _logon_config.tpl // template and field configuration
|   `-- _logon.tpl // logon module
|       `-- _logon_form.tpl
`-- logoff.tpl // log off page, redirects to q.p or homepage



Depending on the dispatch value, _logon_form.tpl is filled with different sub templates:

state::
|== logon_reminder:
|   `-- _logon_form.tpl
|       |-- _logon_reminder_title.tpl
|       |-- _logon_reminder_form.tpl
|       |   `-- _logon_reminder_form_fields.tpl or _logon_reminder_admin_form_fields.tpl // copy from _logon_reminder_form_fields.tpl so that the latter can be customized without messing up admin logon
|       `-- _logon_reminder_support.tpl // backlink to logon form
|== logon_reset:
|   `-- _logon_form.tpl
|       |-- _logon_reset_title.tpl
|       |-- _logon_reset_form.tpl
|       |   `-- _logon_reset_form_fields.tpl
|       `-- _logon_reset_support.tpl // backlink to logon form
|== logon_change:
|   `-- _logon_form.tpl
|       |-- _logon_change_title.tpl
|       |-- _logon_change_form.tpl
|       |   `-- _logon_change_form_fields.tpl
|       `-- _logon_change_support.tpl
`== else:
    `-- _logon_form.tpl
        |== awaiting verification:
        |   `-- _logon_stage.tpl // alternative content for logon box
        `== else:
            |-- _logon_login_title.tpl
            |-- _logon_login_extra.tpl // all-include by other modules
            |-- _logon_error.tpl
            |-- _logon_login_form.tpl
            |   `-- _logon_login_form_fields.tpl
            |-- _logon_login_support.tpl // link forgot password
            |-- _logon_login_outside.tpl
               `-- _logon_link.tpl // all-include by other modules


SIGN UP
=======

signup.tpl // public sign up page, extends base.tpl
|-- _signup_with_email.tpl // staged email signup flow
|   |-- step 1 inline // email address form, postback signup_email_step1
|   |-- _signup_with_email_step2.tpl // inserted by controller_signup
|   |   |-- email code form, postback signup_email_step2
|   |   |-- _email_send_status.tpl // status of email_signup_code.tpl
|   |   |-- logon link for existing local accounts
|   |   `-- external service links for managed domains
|   `-- _signup_with_email_step3.tpl // inserted after a valid email code
|       |-- name fields
|       |-- username/password fields
|       |-- terms checkbox
|       `-- signup errors
`-- _logon_extra.tpl // all-include from external auth modules

controller_signup renders signup.tpl. If the URL contains xs, it fetches the
stored #signup_url{} props and signup_props and passes them to the templates.
The step templates carry those values forward in their wired postbacks.

Email messages and account confirmation:
|-- email_signup_code.tpl // one-time code for step 2
|-- email_verify.tpl // account identity verification link
`-- signup_confirm.tpl // rendered by controller_signup_confirm, extends base.tpl
