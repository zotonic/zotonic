# zotonic_mod_admin

This module includes Bootstrap 5 assets for the admin interface, while keeping the existing Bootstrap 3 build for compatibility.

Included:
- Bootstrap 5 SCSS sources under `priv/lib-src/admin-bootstrap5/bootstrap/scss`.
- Compiled CSS under `priv/lib/css/admin-bootstrap5.css` and `priv/lib/css/admin-bootstrap5.min.css`.
- Bootstrap 3 compatibility shims under `priv/lib-src/admin-bootstrap5/compat` to keep legacy template classes working.
- Existing Bootstrap 3 build under `priv/lib-src/admin-bootstrap3` remains unchanged.
