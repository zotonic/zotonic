.. _url_fetch_options:

url_fetch_options
^^^^^^^^^^^^^^^^^

Determine the URL fetch options for fetching the content of an URL. Used by z_fetch.erl. 


Type: 
    :ref:`notification-first`

Return: 
    ``z_url_fetch:options()``

``#url_fetch_options{}`` properties:
    - host: ``binary``
    - url: ``binary``
    - options: ``z_url_fetch:options()``
