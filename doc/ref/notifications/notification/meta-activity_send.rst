.. _activity_send:

activity_send
^^^^^^^^^^^^^

Push a list of activities via a 'channel' (eg 'email') to a recipient. 
The activities are a list of #activity{} records. 


Type: 
    :ref:`notification-first`

Return: 
    

``#activity_send{}`` properties:
    - recipient_id: ``unknown``
    - channel: ``unknown``
    - queue: ``unknown``
    - activities: ``list``
