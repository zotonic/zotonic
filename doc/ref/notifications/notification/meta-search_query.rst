.. _search_query:

search_query
^^^^^^^^^^^^

Map a custom search term to a ``#search_sql_term{}`` record. 


Type: 
    :ref:`notification-first`

Return: 
    ``#search_sql_term{}``, ``[]``, or ``undefined``

``#search_query{}`` properties:
    - name: ``union``
    - args: ``union``
    - offsetlimit: ``tuple``
    - unknown: ``unknown``
    - search: ``union``
