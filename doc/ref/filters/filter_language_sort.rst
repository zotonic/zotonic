
.. include:: meta-language_sort.rst

Sort a list of language codes or map with languages on their name.
Return a list of ``{Code, LanguageProps}`` pairs.

LanguageProps is a map::

    #{
        fallback => [],
        language => <<"nl">>,
        language_atom => nl,
        name => <<"Nederlands">>,
        name_en => <<"Dutch">>,
        sort_key => <<"dutch">>,
        sublanguages => [
        ]
    }
