Description

Follow users on Twitter using the streaming HTTP API.

Setup instructions: * Enable the mod_twitter module * Put your login/password in the config keys mod_twitter.api_login and mod_twitter.api_password, respectively. * Create a person in the Zotonic database, find a twitter ID on twitter, and put it in the person record on the admin edit page (sidebar) * The module will start automatically to follow the users which have a twitter id set.
