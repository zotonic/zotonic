Development server.

Copyright © 2009 Marc Worrell Date: 2009-11-01

Behaviours: gen_server.

Authors: Marc Worrell (marc@worrell.nl), Bryan Fink (bryan@basho.com).

Description

Development server. Periodically performs a "make" and loads new files. When new files are loaded the caches are emptied.

terminate/2

terminate(Reason, State) -> void()

This function is called by a gen_server when it is about to terminate. It should be the opposite of Module:init/1 and do any necessary cleaning up. When it returns, the gen_server terminates with Reason. The return value is ignored.
