Copyright © 2009 Marc Worrell Date: 2009-07-22

Behaviours: gen_server.

Authors: Marc Worrell (marc@worrell.nl).

Description

Server to execute command line resizes. Prevents competing resize processes when an image on a "hot" page is replaced. Probably the smallest server built in a long while.
